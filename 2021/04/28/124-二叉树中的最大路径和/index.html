124. 二叉树中的最大路径和

发表于 2021-04-28 | 分类于 leetcode

难度hard
路径 被定义为一条从树中任意节点出发，沿父节点-子节点连接，达到任意节点的序列。同一个节点在一条路径序列中 至多出现一次 。该路径 至少包含一个 节点，且不一定经过根节点。

路径和 是路径中各节点值的总和。

给你一个二叉树的根节点 root ，返回其 最大路径和 。

示例 1：

输入：root = [1,2,3]
输出：6
解释：最优路径是 2 -> 1 -> 3 ，路径和为 2 + 1 + 3 = 6
示例 2：

输入：root = [-10,9,20,null,null,15,7]
输出：42
解释：最优路径是 15 -> 20 -> 7 ，路径和为 15 + 20 + 7 = 42

提示：

树中节点数目范围是 [1, 3 * 104]
-1000 <= Node.val <= 1000

解题思路：这道题看了grandyang的题解，grandyang也是看了别人的，我把思路说一下，这种树的问题一般还是用递归回溯的方式来求解的，我们在递归函数里求解包含某个结点的路径和的最大值，最大值就是当前这个结点的值加上以左右字节点为一个端点的路径的最大值，当然这个最大值需要是正值，如果是负值还不如不加，因此我们用一个变量res来更新递归遍历的过程中出现的最大值，然后递归函数的返回值就是以当前结点为一个端点的路径的最大值，这里是一个端点，因此需要在左右子树中取一个较大值，然后加上当前节点值，返回。返回给上一层，恰好就是以当前结点为端点的路径的最大值，可以为上一层所用。这个过程注意的点是，res是动态更新的，但递归函数返回的不是直接结果，而是结果的一部分，需要利用这个部分来拼出新的结果。
