1144. 递减元素使数组呈锯齿状

发表于 2021-04-16 | 分类于 leetcode

难度 medium
给你一个整数数组 nums，每次 操作 会从中选择一个元素并 将该元素的值减少 1。

如果符合下列情况之一，则数组 A 就是 锯齿数组：

每个偶数索引对应的元素都大于相邻的元素，即 A[0] > A[1] < A[2] > A[3] < A[4] > …
或者，每个奇数索引对应的元素都大于相邻的元素，即 A[0] < A[1] > A[2] < A[3] > A[4] < …
返回将数组 nums 转换为锯齿数组所需的最小操作次数。

示例 1：

输入：nums = [1,2,3]
输出：2
解释：我们可以把 2 递减到 0，或把 3 递减到 1。
示例 2：

输入：nums = [9,6,1,6,2]
输出：4

提示：

1 <= nums.length <= 1000
1 <= nums[i] <= 1000

解题思路：这里的思路就是把奇偶数索引分开，对于数组中的元素i，考虑将其递减到比左右元素都更小，这样就是“锯齿”了，因此，如果元素i比左右元素都小，那递减操作为0,不然的话，递减操作就是在元素i减去左右两个元素中的最小值再加1，也就是经过这么多步运算可以把当前元素降到比左右元素都小，最后遍历求和即可。
