98. 验证二叉搜索树

发表于 2021-04-29 | 分类于 leetcode

难度 medium
给定一个二叉树，判断其是否是一个有效的二叉搜索树。

假设一个二叉搜索树具有如下特征：

节点的左子树只包含小于当前节点的数。
节点的右子树只包含大于当前节点的数。
所有左子树和右子树自身必须也是二叉搜索树。

输入:

输出: true
示例 2:

输入:

输出: false
解释: 输入为: [5,1,4,null,null,3,6]。
根节点的值为 5 ，但是其右子节点值为 4 。

解题思路1：这种解题思路是较为简单的，还是用递归的思路解决的，对于每个节点，我们需要确定一个最大值最小值范围，当node的值超出这个范围，就是错的，否则就对左子树和右子树分别调用递归函数，左子树的允许最小值不变，和当前节点相同，允许最大值变为当前节点的值，右子树则正好相反，允许最大值不变，允许最小值变为当前节点值，因为我们明确树中节点的值为int类型，我们为了避免出现错误，将允许最大值最小值设置为Long类型的MAX_VALUE和MIN_VALUE，这样就保证在int取边界值的时候不会出错。

解题思路2：还可用中序遍历把节点先保存到数组里，再遍历数组看是否有后面的数小于等于前面的数的情况，如果有，返回false，否则返回true。
代码 t28 s67 java

解题思路3：还是用上面的思路，中序遍历树，但同时记录上一个遍历到的节点的值，后遍历的节点的值必须大于前一个节点的值，否则就返回false。遍历完没有返回的话，再返回true，这里用的非递归的方式进行中序遍历，同时如果树不满足二叉搜索树的定义，不必遍历完整棵树，而是在中间返回。
