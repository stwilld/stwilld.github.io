828. 统计子串中的唯一字符

发表于 2021-06-18 | 分类于 leetcode

难度 hard
我们定义了一个函数 countUniqueChars(s) 来统计字符串 s 中的唯一字符，并返回唯一字符的个数。

例如：s = “LEETCODE” ，则其中 “L”, “T”,”C”,”O”,”D” 都是唯一字符，因为它们只出现一次，所以 countUniqueChars(s) = 5 。

本题将会给你一个字符串 s ，我们需要返回 countUniqueChars(t) 的总和，其中 t 是 s 的子字符串。注意，某些子字符串可能是重复的，但你统计时也必须算上这些重复的子字符串（也就是说，你必须统计 s 的所有子字符串中的唯一字符）。

由于答案可能非常大，请将结果 mod 10 ^ 9 + 7 后再返回。

示例 1：

输入: s = “ABC”
输出: 10
解释: 所有可能的子串为：”A”,”B”,”C”,”AB”,”BC” 和 “ABC”。
其中，每一个子串都由独特字符构成。
所以其长度总和为：1 + 1 + 1 + 2 + 2 + 3 = 10
示例 2：

输入: s = “ABA”
输出: 8
解释: 除了 countUniqueChars(“ABA”) = 1 之外，其余与示例 1 相同。
示例 3：

输入：s = “LEETCODE”
输出：92

提示：

0 <= s.length <= 10^4
s 只包含大写英文字符

解题思路：这道题看了leetcode上的题解，思路还是很巧妙的，简单地说下，先找出某个字母仅出现一次的区间，以”LEETCODE”中的”T”为例，其出现的区间为[0,7]，然后开始统计这个区间内”T”在多少子串中出现过，以左边字符”L”,”E”,”E”,”T”开头的情况有4种，以右边字符”T”,”C”,”O”,”D”,”E”结尾的情况有5种，因此”T”在45总共20个子串中出现过，对最后总次数的贡献也即是20。而已第二个字符”E”为例，其仅出现一次的区间为[0,1],这个区间以”L”,”E”开头的情况有2种，以”E”结尾的情况有1种，因此”E”总共在21共2个子串中出现过。知道以上技巧后，下面就是要找到每个字母仅出现一次的区间了，这里用两个数组left和right表示字母仅出现一次的区间的左右端点，然后我们用一个数组mem来记录每个字母出现的位置。先从左到右遍历字符串，索引为i，mem初始化为-1，表示最左的端点，算出某个字符的下标t后，我们先将left[i]赋为mem[t]，然后再将mem[t]赋为i，这是为了保证i处的字符先把之前记录下来的与本身相等的位置确定为左端点，然后再把新的位置赋给mem[i]，作为下次出现相同字符时的右端点，以”LEETCODE”为例，遍历到第二个字符”E”的时候，left[1]=-1,而随后mem[‘E’-‘A’]被赋为1，因此遍历到第三个字符的时候，left[2]=1,这样才能保证左端点正确更新。right数组的赋值同理，但此时要把mem数组先初始化为字符串长度len，表示默认的右端点是len。最后，统计结果时，把i距离左右端点的距离相乘，加到res中即可，记得对1000000007取模即可。

代码 t75 s74 java
