483. 最小好进制

发表于 2021-06-18 | 分类于 leetcode

难度 hard
对于给定的整数 n, 如果n的k（k>=2）进制数的所有数位全为1，则称 k（k>=2）是 n 的一个好进制。

以字符串的形式给出 n, 以字符串的形式返回 n 的最小好进制。

示例 1：

输入：”13”
输出：”3”
解释：13 的 3 进制是 111。
示例 2：

输入：”4681”
输出：”8”
解释：4681 的 8 进制是 11111。
示例 3：

输入：”1000000000000000000”
输出：”999999999999999999”
解释：1000000000000000000 的 999999999999999999 进制是 11。

提示：

n的取值范围是 [3, 10^18]。
输入总是有效且没有前导 0。

解题思路：这道题目需要用到数学的一些技巧，首先，我们最后要得到的结果是$(11…11)k=(n){10}$，也就是k进制的11…11换算成十进制的n，我们假设总共有m位，则有 $x^{m-1} + x^{m-2} + … + x^2 + x + 1=n$，由此有$x^{m-1}<n$，又$(x+1)^m = (x+1)^{m-1} + a*(x+1)^{m-2} + … + b*x + 1>x^{m-1} + x^{m-2} + … + x^2 + x + 1=n$，则有$(x+1)^{m-1}>n$，即$x<n^{1/(m-1)}<x+1$，由于我们的目的是为了求出一个最小的好进制，即$x$尽可能小，因此$m$要尽可能大，又$2^{59}=1.15*10^{18}>10^{18}$，所以我们最多也就取到59，因为二进制就是最小的了，所以我们从59向1遍历，然后用$n^{1/(m-1)}$取一个值，作为$x$的候选，如果x<=1,那直接跳过，如果满足$x^{m-1} + x^{m-2} + … + x^2 + x + 1=n$，此时$x$就是满足要求的数，直接返回，否则的话就是返回$n-1$,这种情况下$n-1$进制下的(11)是满足条件的。
