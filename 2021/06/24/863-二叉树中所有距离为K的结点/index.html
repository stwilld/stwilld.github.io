863. 二叉树中所有距离为 K 的结点

发表于 2021-06-24 | 分类于 leetcode

难度 medium
给定一个二叉树（具有根结点 root）， 一个目标结点 target ，和一个整数值 K 。

返回到目标结点 target 距离为 K 的所有结点的值的列表。 答案可以以任何顺序返回。

示例 1：

输入：root = [3,5,1,6,2,0,8,null,null,7,4], target = 5, K = 2
输出：[7,4,1]
解释：
所求结点为与目标结点（值为 5）距离为 2 的结点，
值分别为 7，4，以及 1

注意，输入的 “root” 和 “target” 实际上是树上的结点。
上面的输入仅仅是对这些对象进行了序列化描述。

提示：

给定的树是非空的。
树上的每个结点都具有唯一的值 0 <= node.val <= 500 。
目标结点 target 是树上的结点。
0 <= K <= 1000.

解题思路：用一个hashmap建立子节点到父节点的映射，然后从根节点开始，进行递归。在递归函数中，首先用一个hashset来记录已经访问过的节点，然后判断当前K是否为0（K在递归的过程中递减），如果K=0，说明当前节点符合条件，直接添加到res中，如果不符合，那调用递归，在实参中，K要减一，分别对左子节点，右子节点，还有hashmap中存在的父节点进行递归。
