472-连接词

发表于 2021-12-28 | 分类于 leetcode

难度 hard

链接：https://leetcode-cn.com/problems/concatenated-words

题目描述：

给你一个 不含重复 单词的字符串数组 words ，请你找出并返回 words 中的所有 连接词 。

连接词 定义为：一个完全由给定数组中的至少两个较短单词组成的字符串。

示例 1：

输入：words = [“cat”,”cats”,”catsdogcats”,”dog”,”dogcatsdog”,”hippopotamuses”,”rat”,”ratcatdogcat”]
输出：[“catsdogcats”,”dogcatsdog”,”ratcatdogcat”]
解释：”catsdogcats” 由 “cats”, “dog” 和 “cats” 组成;
“dogcatsdog” 由 “dog”, “cats” 和 “dog” 组成;
“ratcatdogcat” 由 “rat”, “cat”, “dog” 和 “cat” 组成。
示例 2：

输入：words = [“cat”,”dog”,”catdog”]
输出：[“catdog”]

提示：

1 <= words.length <= 104
0 <= words[i].length <= 1000
words[i] 仅由小写字母组成
0 <= sum(words[i].length) <= 105

解题思路：

这道题目用的是“字符串哈希+动态规划”的解法，参考了宫水三叶的题解。字符串哈希是用一个数值来表示一个字符串，在找子字符的时候可以根据数值是否相等进行匹配。这里将words中的所有单词，都进行了字符串哈希，然后保存到一个HashSet里面，其类型为Long。接下来就是遍历words中的所有单词，看某个单词是否完全由words中的其他单词组成，如果是，则加入结果List中。判断某个单词是否由words中的其他单词组成采用的是动态规划的算法，假设当前遍历的单词为word，其长度为len，我们用一个长度为len+1的int数组f来记录状态，其中f[i]表示word[0:i-1]这一段字符中包含的words中其他单词的数量，当f[len]>1时，表示word中包含大于1个words中其他单词的数量，也就是word满足条件，应该加入结果List中。我们用-1填充f数组，并将f[0]置为0，然后i从0到len进行遍历，表示当前遍历到word的第i-1个字符，在循环中首先判断，如果f[i]=-1，则说明到i-1这个位置，word中都没有匹配到words中的任何单词，则直接跳过当前循环，否则开始i-1之后的匹配：用j从i+1到len循环，计算后面字符的哈希值，如果哈希值在HashSet中存在，那说明中间某段字符是words中的某个单词，满足要求，此时用f[j] = Math.max(f[j], f[i] + 1)更新f[j]处的值。当遍历完整个word之后，如果f[len]>1，说明当前word至少由2个words中的单词组成，符合题意，则应当加入结果List中。
