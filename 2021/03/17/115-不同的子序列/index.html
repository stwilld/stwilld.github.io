115. 不同的子序列

发表于 2021-03-17 | 分类于 leetcode

给定一个字符串 s 和一个字符串 t ，计算在 s 的子序列中 t 出现的个数。

字符串的一个 子序列 是指，通过删除一些（也可以不删除）字符且不干扰剩余字符相对位置所组成的新字符串。（例如，”ACE” 是 “ABCDE” 的一个子序列，而 “AEC” 不是）

题目数据保证答案符合 32 位带符号整数范围。

示例 1：

输入：s = “rabbbit”, t = “rabbit”
输出：3
解释：

(上箭头符号 ^ 表示选取的字母)

示例 2：

输入：s = “babgbag”, t = “bag”
输出：5
解释：
如下图所示, 有 5 种可以从 s 中得到 “bag” 的方案。
(上箭头符号 ^ 表示选取的字母)

提示：

0 <= s.length, t.length <= 1000
s 和 t 由英文字母组成

解题思路：这道题一开始看了grandyang的题解115. Distinct Subsequences 不同的子序列，第二次做的时候发现还是不会，后来仔细想了想，想明白了，就写了出来。这道题还是用的动态规划的思路解决问题，dp[i][j]表示的是字符串s前j个字符中包含字符串t前i个字符的数目，注意这里用的是数目，但是等会我们要比较字符是否相等的话用的是下标，也就是数目减1。接下来推导状态转移方程，对于dp[i][j]，它的状态可以如此更新：我们以字符串t的遍历为外循环，已经比较了遍历了字符串s的前j-1个字符，当前我们要确定dp[i][j]，关键在比较t[i-1]和s[j-1]，假设二者不等，那么我们只能以在内循环的上一个状态dp[i][j-1]加上0作为当前状态，表示没有增加任何新的case；如果二者相等，由于dp[i][j-1]以及确定了s的前j-1个字符中包含t前i个字符的情况，所以我们只要再加上把当前s的第j个字符考虑进来时新增的情况，由于此时s[j-1]=t[i-1]，所以新增的情况其实就是dp[i-1][j-1]，因为相当于此时两个字符串都去掉了最后一个相等的字符，把这个新增的情况加到原来的基础上就可以。也就是状态转移方程为：
dp[i][j] = dp[i][j-1] + (s[j-1]==t[i=1] ? dp[i-1][j-1] : 0);
