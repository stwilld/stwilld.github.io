23. 合并K个升序链表

发表于 2021-03-20 | 分类于 leetcode

题目描述：

给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

示例 1：

输入：lists = [[1,4,5],[1,3,4],[2,6]]
输出：[1,1,2,3,4,4,5,6]
解释：链表数组如下：
[
1->4->5,
1->3->4,
2->6
]
将它们合并到一个有序链表中得到。
1->1->2->3->4->4->5->6
示例 2：

输入：lists = []
输出：[]
示例 3：

输入：lists = [[]]
输出：[]

提示：

k == lists.length
0 <= k <= 10^4
0 <= lists[i].length <= 500
-10^4 <= lists[i][j] <= 10^4
lists[i] 按 升序 排列
lists[i].length 的总和不超过 10^4

难度：hard
解题思路: 这道题看的题解，用一个优先队列（最小堆）把每个链表的头都放进去，然后开始把堆顶的元素添加到结果中，如果堆顶的元素有下一个元素的话（next所指元素不为空），那么就把下一个元素放到队列中。这里思路理解起来很简单，还是java容器的问题，不太熟练。
代码 t66 s66 java
