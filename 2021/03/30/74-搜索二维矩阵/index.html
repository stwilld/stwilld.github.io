74. 搜索二维矩阵

发表于 2021-03-30 | 分类于 leetcode

难度 medium
编写一个高效的算法来判断 m x n 矩阵中，是否存在一个目标值。该矩阵具有如下特性：

每行中的整数从左到右按升序排列。
每行的第一个整数大于前一行的最后一个整数。

示例 1：

输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
输出：true
示例 2：

输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13
输出：false

提示：

m == matrix.length
n == matrix[i].length
1 <= m, n <= 100
-104 <= matrix[i][j], target <= 104

解题思路：这里思路相对比较简洁，就是用了两次二分查找，首先对第一列的数进行查找，找到第一个大于target的数，并且将其减一作为横向查找的行索引，然后在那一行继续做二分查找。主要是二分查找的写法，要记住。

代码 t100 s25 java
