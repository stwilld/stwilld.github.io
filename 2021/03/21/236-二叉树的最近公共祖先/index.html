236. 二叉树的最近公共祖先

发表于 2021-03-21 | 分类于 leetcode

难度 medium
给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

示例 1：

输入：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 1
输出：3
解释：节点 5 和节点 1 的最近公共祖先是节点 3 。
示例 2：

输入：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 4
输出：5
解释：节点 5 和节点 4 的最近公共祖先是节点 5 。因为根据定义最近公共祖先节点可以为节点本身。
示例 3：

输入：root = [1,2], p = 1, q = 2
输出：1

提示：

树中节点数目在范围 [2, 105] 内。
-109 <= Node.val <= 109
所有 Node.val 互不相同 。
p != q
p 和 q 均存在于给定的二叉树中。

解题思路：这道题需要用到递归的思想来解决问题，如果当前节点为空或者当前节点等于p或q，那直接返回当前节点，为什么呢？返回null主要是在递归中需要用到，而当当前节点为p或q时，由于p和q在树中一定存在，因此当前节点肯定是最小祖先了。接下来对左右子节点调用递归函数，分别返回left和right，如果返回值均不为空，说明p和q分别位于两个子树中，因此返回当前节点，如果left不为空，即p、q都在左子树中，有两种情况，一种是left会返回p和q中较高的位置，也就是在左子树中p和q的最小祖先是它们自身某个节点，此时right为null，第二种情况是left返回p和q的最小祖先，此时right还是为null，因此我们要返回非空的left，在右子树中的情况也是一样的，不再赘述。记住遍历子树的时候，其实其行为和在遍历父节点的时候是一样的。
