22. 括号生成

发表于 2021-03-27 | 分类于 leetcode

难度 hard

题目描述：

数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

示例 1：

输入：n = 3
输出：[“((()))”,”(()())”,”(())()”,”()(())”,”()()()”]
示例 2：

输入：n = 1
输出：[“()”]

提示：

1 <= n <= 8

解题思路：这道题用的动态规划的思路，也就是对于数字n，我们假定已经知道n-1的括号的排列，也就是我们已经知道所有i<n时的排列情况，这些排列当然就是我们在递推的过程中得到的。对于i=n时，我们假定最左边的左括号是最后添加上去的，这个括号必定有一个与其对应的右括号，然后剩下的n-1个括号，有可能分布在外边，也就是右括号的右边，也有可能分布在括号的内部，也就是我们把前面n-1个括号要放在这两个位置，显然这个时候需要我们进行遍历，并从动态数组的前n-1个数组中获取前n-1组排列。即我们需要遍历dp[p]和dp[q]，其中p+q=n-1，新增的有n个括号的字符串的格式为:”(“+p[idx1]+”)”+q[idx2]，最后说一下为什么我们可以这样假设，其实我们这样假设只是为了方便遍历，这样我们固定了一个点（左括号所在的位置），只需要根据右括号的位置进行遍历，也就是确定剩下n-1个括号要怎么放的问题就可以了，这样简化了问题。我们把问题抽象来看，其实就是在新增的一个括号” ( ) “的三个空隙放入前n-1个括号而已，而括号放在最左边的空隙和放在最右边的空隙这种情况是重复的，所以我们固定好了左括号，也就是避免了这种重复的情况。

代码 t5 s12 java
