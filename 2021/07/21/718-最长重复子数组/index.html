718. 最长重复子数组

发表于 2021-07-21 | 分类于 leetcode

难度 medium
给两个整数数组 A 和 B ，返回两个数组中公共的、长度最长的子数组的长度。

示例：

输入：
A: [1,2,3,2,1]
B: [3,2,1,4,7]
输出：3
解释：
长度最长的公共子数组是 [3, 2, 1] 。

提示：

1 <= len(A), len(B) <= 1000
0 <= A[i], B[i] < 100

解题思路：这道题目和1143. 最长公共子序列这个题目属于同一类型，一个是求最大公共子序列，一个是求最大公共子串，可以参考另一道题目的做法。这道题目用了一个二维dp数组来解决，我们用两层循环i，j来遍历，然后如果nums1[i]==nums2[j]，相应的就有dp[i+1][j+1]=dp[i][j]+1，然后用一个值res动态更新到当前为止的最大值，最后返回即可。
