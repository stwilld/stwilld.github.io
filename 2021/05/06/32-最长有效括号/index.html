32. 最长有效括号

发表于 2021-05-06 | 分类于 leetcode

难度hard
给你一个只包含 ‘(‘ 和 ‘)’ 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

示例 1：

输入：s = “(()”
输出：2
解释：最长有效括号子串是 “()”
示例 2：

输入：s = “)()())”
输出：4
解释：最长有效括号子串是 “()()”
示例 3：

输入：s = “”
输出：0

提示：

0 <= s.length <= 3 * 104
s[i] 为 ‘(‘ 或 ‘)’

解题思路：grandyang
这里还是借助栈来求解，需要定义一个 start 变量来记录合法括号串的起始位置，遍历字符串，如果遇到左括号，则将当前下标压入栈，如果遇到右括号，如果当前栈为空，说明能够完整匹配的一段已经匹配完了，需要另起一段，则将下一个坐标位置记录到 start（也就是start=i+1)，这里其实就在寻找新的字符串的可能起点，而如果栈不为空，则将栈顶元素取出，此时有两种情况，一种情况是栈已经空了，这样我们就需要借助之前记录下来的start值，来更新最大匹配长度，则res更新为res和 i - start + 1 中的较大值；另一种情况就是栈还没空，说明还没把左括号匹配完，只要把此时的栈顶元素作为匹配段的起点即可，因为从栈顶元素到当前索引中间那部分都是能匹配的，因此res更新为res和 i - st.top() 中的较大值。
